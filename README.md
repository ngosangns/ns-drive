# ns-drive

## Prerequisites

- Wails 2
- Go v1.23.4
- Taskfile

## Build Commands

### Windows

```sh
task build-win
```

### Mac

```sh
task build-mac
```

## Execution

Execute the `./desktop.exe` file located in the project's root directory after building.
